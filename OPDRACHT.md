## Opdrachtomschrijving
Maak een simpele Laravel Blog applicatie met een overzicht/detail constructie gevoed door een API.

- Gebouwd in Laravel icm. Blade templates
- Data opgeslagen in MySQL / MariaDB database
- Vulling van de database (posts / comments)  middels een API, via de mock API op: - https://jsonplaceholder.typicode.com/
- Bestaande uit:
    - Achtergrond taken voor vullen van database via de API
    - Overzichtspagina
        - Titel van de post
        - Auteur
        - Eerste 50 woorden van de body
        - Link naar detail
        - Nette SEO vriendelijke URL
        - Inclusief paginering
    - Detailpagina
        - Titel van de post
        - Auteur
        - Volledige body
        - Comments behorende bij de post
        - Nette SEO vriendelijke URL obv titel van de post
        - Mogelijk maken om extra comments te kunnen plaatsen middels een formulier onderaan de pagina (Naam / Email / Bericht)


## tijdsduur / oplevering
Deze opdracht binnen 4.5 uur maken
